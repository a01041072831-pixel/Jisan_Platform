# 06. 최종 품질 검증 체크리스트

작성이 완료된 손해사정서는 제출 전 아래 **모든 항목**을 검증합니다.
**단 하나의 항목이라도 불완전하면 수정 후 재검증합니다.**

---

## 1. 사실관계 정확성

- [ ] 모든 인적사항(이름, 생년월일, 주소 등)이 정확히 기재되었는가?
- [ ] 보험계약사항(증권번호, 보험기간, 담보내역)이 원본과 일치하는가?
- [ ] 사고내용과 경위가 제공된 자료와 정확히 일치하는가?
- [ ] 의료기록 및 진단명이 정확히 기재되었는가?

## 2. 논리적 일관성

- [ ] 손해사정 의견이 제시된 사실과 논리적으로 일관성이 있는가?
- [ ] 보험약관의 해석과 적용이 일관되게 이루어졌는가?
- [ ] 의학적 소견과 장해율 적용이 합리적으로 연결되는가?

## 3. 계산 정확성

- [ ] 보험금 계산에 오류는 없는가?
- [ ] 적용된 장해율이 약관 기준에 맞게 적용되었는가?
- [ ] 합계 금액이 정확히 계산되었는가?

## 4. 법적 적합성

- [ ] 인용된 법규와 약관 조항이 최신 버전인가?
- [ ] 보험업법 및 감독규정에 위배되는 내용은 없는가?
- [ ] 판례 인용이 정확하고 적절한가?

## 5. 형식적 완결성

- [ ] 모든 필수 섹션(Ⅰ~Ⅵ)이 포함되어 있는가?
- [ ] 첫 페이지 공문 형식이 올바른가?
- [ ] 표와 단락이 페이지 경계에서 분리되지 않았는가?
- [ ] 날짜, 서명 등 형식적 요소가 모두 갖추어졌는가?
- [ ] 페이지 나누기(`page-break-before`)가 적절히 삽입되었는가?

## 6. 할루시네이션 검증

- [ ] 제공되지 않은 정보가 임의로 추가되지 않았는가?
- [ ] 모든 내용이 제공된 자료에 근거하고 있는가?
- [ ] 불확실한 정보는 "정보 미제공"으로 적절히 표시되었는가?
- [ ] 각 사실 진술의 출처가 추적 가능한가?
